## Кратко о документе
Provisioning в management & orchestration.

## Связь с 3GPP NMS
описывает логику предоставления и настройки ресурсов/сервисов.

## Чем полезен на практике
хорошая база для API и workflow по созданию/обновлению сервисных сущностей.

## Для какого функционала
provisioning API, шаблоны операций, статусные процессы.

## Что изучить в документе
- принципы provisioning в management контуре;
- операции создания/обновления ресурсов;
- формат статусов и результатов операций.

## Как применить в проекте
- использовать для расширения групповых операций;
- согласовать API provision-запросов.

## Что подготовить после чтения
- шаблон provision операции;
- таблица статусов выполнения.
