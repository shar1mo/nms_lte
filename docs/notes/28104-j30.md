## Кратко о документе
Management Data Analytics (MDA).

## Связь с 3GPP NMS
определяет место аналитики в контуре управления.

## Чем полезен на практике
база для этапа 2, когда после базового мониторинга нужна интеллектуальная аналитика.

## Для какого функционала
MDA отчеты, корреляции, продвинутые рекомендации.

## Что изучить в документе
- структуру MDA-функций;
- форматы аналитических отчетов и входных данных;
- требования к качеству и полноте данных для аналитики.

## Как применить в проекте
- готовить расширение после MVP без ломки архитектуры;
- проектировать API так, чтобы можно было добавить аналитические отчеты.

## Что подготовить после чтения
- список MDA-сценариев на этап 2;
- требования к данным для последующей аналитики.
