## Кратко о документе
5G NRM Stage 2/3 (детализированные классы и атрибуты).

## Связь с 3GPP NMS
основной источник для практической реализации модели ресурсов.

## Чем полезен на практике
напрямую используется в inventory, топологии и CM.

## Для какого функционала
структура данных NE, межобъектные связи, профиль поддерживаемых сущностей.

## Что изучить в документе
- детальные IOC и атрибуты 5G NRM;
- связи между объектами и контейнирование;
- ограничения и особенности реализации.

## Как применить в проекте
- использовать для практического моделирования inventory;
- подготовить профиль поддерживаемых объектов по доменам.

## Что подготовить после чтения
- профиль поддерживаемых IOCs;
- карта "IOC -> API -> хранилище".
