- **Файл:** `docs/specs/28622-j60.pdf`
- **Спецификация:** `TS 28.622`
- **Приоритет:** `P1`
- **Кому читать:** `Оба`
- **Неделя:** `02`

## Кратко о документе
Generic NRM Information Service.

## Связь с 3GPP NMS
базовая информационная модель для управления.

## Чем полезен на практике
фундамент для `ManagedElement`, `SubNetwork`, общих атрибутов и иерархий.

## Для какого функционала
общая модель inventory, базовые классы, унификация структуры объектов.

## Что изучить в документе
- базовые классы Generic NRM IS;
- структуру иерархии управляемых объектов;
- общие атрибуты и принципы контейнирования.

## Как применить в проекте
- использовать как основу inventory-модели;
- держать единый формат представления объектов в API.

## Что подготовить после чтения
- минимальная generic модель для проекта;
- список обязательных полей объектов.
