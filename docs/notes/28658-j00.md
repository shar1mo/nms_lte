# Заметка: `28658-j00.pdf`

- **Файл:** `docs/specs/28658-j00.pdf`
- **Спецификация:** `TS 28.658`
- **Приоритет:** `P1`
- **Кому читать:** `Оба`
- **Неделя:** `02`

## Кратко о документе
E-UTRAN NRM Information Service.

## Связь с 3GPP NMS
ключевой LTE документ по объектам и отношениям в управляемой модели.

## Чем полезен на практике
источник истины для LTE inventory и CM модели (`ENBFunction`, `EUtranCell*`, `EUtranFrequency`, `EUtranRelation`).

## Для какого функционала
LTE-профиль объектов MVP, данные инвентаря, валидация CM scope.

## Что изучить в документе
- набор LTE IOCs и их атрибуты;
- отношения между LTE объектами;
- ограничения и уведомления для E-UTRAN модели.

## Как применить в проекте
- использовать как основной LTE профиль для MVP;
- проверять, что inventory и CM покрывают ключевые LTE сущности.

## Что подготовить после чтения
- LTE object profile для проекта;
- список поддерживаемых LTE операций в MVP.
