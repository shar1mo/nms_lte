## Кратко о документе
management service template.

## Связь с 3GPP NMS
задает шаблон описания управленческих сервисов.

## Чем полезен на практике
удобная опора для стандартизации внутренних и внешних API сервисов.

## Для какого функционала
проектирование API-контрактов, единый формат описания сервисов.

## Что изучить в документе
- management service template;
- структуру описания операций сервиса;
- единый подход к сервисным контрактам.

## Как применить в проекте
- шаблонизировать описание всех API сервисов;
- избежать разнородных форматов документации.

## Что подготовить после чтения
- единый шаблон описания сервисов;
- актуальные шаблоны для новых API методов.
