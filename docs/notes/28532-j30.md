## Кратко о документе
Generic management services.

## Связь с 3GPP NMS
задает общий каркас сервисов управления.

## Чем полезен на практике
напрямую используется при проектировании northbound API и событий.

## Для какого функционала
общие сервисы управления, heartbeat, generic API шаблоны.

## Что изучить в документе
- набор generic management сервисов;
- общие правила для операций и уведомлений;
- модель повторно используемых сервисных шаблонов.

## Как применить в проекте
- построить единый стиль API и уведомлений;
- свести повторяющиеся операции к общей модели.

## Что подготовить после чтения
- каталог generic API методов;
- единые правила ошибок и ответов.
