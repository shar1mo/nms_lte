# Неделя 16. Подготовка пилота

## Цель недели
Подготовить систему и документы к реальному пилоту.

## Задачи

Студент 1:
- настраивает рабочие параметры таймаутов и повторов;
- фиксирует профиль подключенных узлов.

Студент 2:
- пишет короткую инструкцию по эксплуатации для оператора;
- готовит чек-лист запуска.

## Что читаем
- `docs/specs/23003-j50.pdf`
- `docs/specs/32158-j10.pdf`

## Результат недели
- готовый релиз-кандидат для пилотной среды.

## Проверка
Чек-лист запуска проходит без блокирующих замечаний.
