# Неделя 04. Подключение узлов

## Цель недели
Сделать стабильное подключение узла и хранение его состояния.

## Задачи

Студент 1:
- делает подключение и переподключение;
- сохраняет поддерживаемые возможности узла.

Студент 2:
- делает API добавления/удаления узла;
- хранит статус узла в базе.

## Что читаем
- `docs/specs/28623-j60/_3gpp-common-managed-element.yang`
- `docs/specs/28623-j60/_3gpp-common-subnetwork.yang`
- `docs/specs/28658-j00.pdf`

## Результат недели
- можно добавить тестовый узел и видеть его статус.

## Проверка
После обрыва соединения узел корректно возвращается в рабочее состояние.
