# Неделя 12. Простые групповые изменения

## Цель недели
Сделать минимальный режим изменения конфигурации для небольшой группы узлов.

## Задачи

Студент 1:
- выполняет однотипную операцию по списку узлов;
- добавляет ограничение скорости.

Студент 2:
- показывает результат по каждому узлу;
- формирует простой итоговый отчет.

## Что читаем
- `docs/specs/28530-j00.pdf`
- `docs/specs/28531-j20/TS28531_NSProvMnS.yaml`

## Результат недели

## Проверка
По API видно статус каждого узла в группе.
