# Неделя 03. Каркас сервиса на Go

## Цель недели
Запустить техническую основу проекта.

## Задачи

Студент 1:
- создает запуск API сервиса;
- поднимает модуль работы с `NETCONF`.

Студент 2:
- создает модули `inventory`, `cm`, `fault`, `pm`;
- подключает PostgreSQL и миграции.

## Что читаем
- `docs/specs/32158-j10.pdf`
- `docs/specs/32160-j50.pdf`
- `docs/specs/28623-j60/TS28623_ComDefs.yaml`

## Результат недели
- сервис собирается и запускается локально.

## Проверка
`go build` проходит, метод проверки состояния сервиса работает.
