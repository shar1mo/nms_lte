# Неделя 14. Наблюдаемость

## Цель недели
Сделать простые метрики и дашборды для контроля системы.

## Задачи

Студент 1:
- добавляет метрики по сессиям и ошибкам `NETCONF`;
- унифицирует формат логов.

Студент 2:
- собирает базовые дашборды;
- настраивает простые оповещения.

## Что читаем
- `docs/specs/28537-j40.pdf`
- `docs/specs/28550-j20.pdf`

## Результат недели
- видимость состояния сервиса в реальном времени.

## Проверка
Деградация видна на дашборде без ручного поиска по логам.
