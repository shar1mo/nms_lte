# Неделя 02. Базовая модель LTE

## Цель недели
Собрать минимальную модель LTE объектов, с которой будет работать сервис.

## Задачи

Студент 1:
- выделяет базовые объекты из TS 28.658;
- связывает их с доступными YANG модулями.

Студент 2:
- связывает объекты модели с методами API;
- задает правила идентификаторов (`dn`, `ldn`, внутренний id).

## Что читаем
- `docs/specs/28658-j00.pdf`
- `docs/specs/28659-j00.pdf`
- `docs/specs/28622-j60.pdf`
- `docs/specs/28623-j60/28623-j60.pdf`

## Результат недели
- короткий документ «минимальный LTE профиль MVP».

## Проверка
Для каждого метода API понятно, с каким объектом модели он работает.
