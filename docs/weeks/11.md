# Неделя 11. API версии 1

## Цель недели
Зафиксировать стабильный внешний API.

## Задачи

Студент 1:
- добавляет технические методы API (проверка состояния сервиса, проверка готовности сервиса, информация о возможностях узла);
- стандартизует технические ошибки.

Студент 2:
- описывает OpenAPI v1;
- проверяет основные методы API контрактными тестами.
