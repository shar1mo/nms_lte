# Неделя 13. Минимальная безопасность

## Цель недели
Добавить базовую защиту для пилотного запуска.

## Задачи

Студент 1:
- настраивает безопасное хранение ключей;
- убирает небезопасные параметры подключения.

Студент 2:
- добавляет простые роли доступа;
- включает журнал действий пользователей.

## Что читаем
- `docs/specs/28500-j00.pdf`
- `docs/specs/28510-j00.pdf`
- `docs/specs/32158-j10.pdf`

## Результат недели
- работающая базовая модель доступа.

## Проверка
Пользователь без прав не может запускать изменение конфигурации.
